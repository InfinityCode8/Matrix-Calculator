UK
The program must be able to perform the following functions:

a) input data into the system can be entered in two ways
The first way is to enter the data yourself (number of rows "m",
columns "n" and the coefficients of the system "a1a1, a1a2,… a1an" 
with free terms "b1, b2,… bn", the minimum size of the matrix coefficients 2x2,
the maximum size 4x4 , the number of free members of the matrix corresponds to the number of rows of this matrix);

The second way is to enter data through a file (in the file itself you must first enter the size of the square matrix,
one number, in the next paragraph to enter the coefficients of the system through ";" after the coefficients is a free member,
followed by "." 4x4, the number of free members of the matrix corresponds to the number of rows of this matrix);

b) the program should show the stages of solution of the system of linear equations and the result itself 
(the accuracy of calculating the solution of the system of linear equations has an accuracy after zero three digits, 0.0234);

c) the program provides a choice of three methods for solving a system of linear equations, namely, the Gaussian Method, the Cramer Method and the Matrix Method.



UA
Програма повинна забезпечувати можливість виконання наступних перелічених функцій:

а) вхідні дані у систему можна вводити двома способами
Перший спосіб, вводити дані самостійно (кількість рядків “m”, стовпчиків “n” і 
самі коефіціенти системи “а1а1, a1a2, … a1an” з вільними членами “b1, b2, … bn ”,
мінімальний розмір коефіціетів матриці 2х2, максимальний розмір 4х4, кількість 
вільних членів матриці відповідають кількості рядів цієї матриці); 
Другий спосіб, вводити дані через файл (у самому файлі потрібно спочатку вводити розмір квадратної матриці,
один числом, на настопному абзаці вводити коефіенти системи через «;» після коефіціентів йде вільний член,
після нього ставиться «.», мінімальний розмір матриці 2х2, максимальний 4х4, кількість вільних членів матриці відповідають кількості рядів цієї матриці);

б) програма має показувати етапи розв’язку системи лінійних рівнянь і сам результат   
(точність обчислення розв’язку системи лінійних рівнянь має точність після нуля три цифри, 0,0234);

в) програма надає для розв’язку системи лінійних рівнянь на вибір три методи а саме, Метод Гауса, Метод Крамера і Матричний метод.
